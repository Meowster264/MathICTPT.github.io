DEFINITIONS: Terms you must learn

• Relation - A set of ordered pairs
• Domain - A set of all first coordinates (x-coordinates) from the ordered pairs.
• Range - A set of all second coordinates (y-coordinates) from the ordered pairs.
• Graph - The set of points in the coordinate plane

Extra:
- {} are the symbols when writing sets
- When writing the domain and range, we must not repeat the values

Many to One:
A function is called 'many to one' if some function output value corresponds to more than one input value.

One to Many:
A one to many is always just a RELATION, not a function.
